# File Management System 
A simple Windows Forms file browser with preview support.
## Features
CRUD operations
Browse folders in a tree view
Preview images (jpg, png, gif, bmp, ico)
Preview text files (txt, cs, json, xml, html, css, js, md, log)

## Requirements

.NET 10.0
Windows

## Usage

Open in Visual Studio
Press F5 to run
Click folders to expand, click files to preview

## Implementation Guidelines
##GUI Components
File/Directory Display
A tree view showing files and folders in a hierarchical structure.
Action Buttons
Buttons for Create, Open, Save, Delete, Rename, and Refresh operations.
Current Path Display
Text box at the top showing the full path of the selected item.
Status/Feedback Area
Status bar at the bottom displaying operation results and file information.
File Content Area
Split panel with image preview (PictureBox) and text preview (TextBox).
